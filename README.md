# Student Management System (Java)

A console-based Student Management System built using Core Java and OOP concepts.

## Features

- Add Student
- View Students
- Update Student
- Delete Student
- Partial Search
- Sort by Roll Number
- File Handling
- Data Persistence
- Validation
- Exception Handling
- Interfaces
- Abstract Classes
- Inheritance
- Polymorphism

## Technologies

- Java
- OOP
- Collections Framework
- File Handling
